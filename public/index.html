API documentation

  • GET / = root info page
  • GET /books = get all books
  • GET /books:?id=[id] = get book with specific id
  • GET /users:?user=[user] = get user info with specific username
  • POST /books = post books with given body
  • DELETE /books:id?id=[id] = delete book with specific id
  • PATCH /books:id?id=[id] = update book with specific id
